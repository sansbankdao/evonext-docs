## Getting Started

Ready to claim your unique spot on the Dash Platform? Let's get started with a simple process that puts you in control of your digital identity.

### Prerequisites

:::info What you'll need

- [__Your 12/24 word seed phrase__](https://www.coinbase.com/learn/wallet/what-is-a-seed-phrase) to connect with EvoNext and begin your identity activities:
    - Register a NEW Platform Identity (if you don't have one)
    - Register a NEW Platform Username
    - Create a NEW EvoNext Profile

- A device with EvoNext installed or access to the web application
- Sufficient Dash balance to cover network fees (typically less than 0.01 Dash)

:::

## Identity + Username Registration

If you're new to the Dash Platform, you'll need to create an identity before claiming a username. Here's how to get both set up in one seamless process:

:::tip Quick Start Option

The fastest way to get started is by using EvoNext, which will guide you through both identity creation and username registration in a user-friendly interface.

:::

### Step-by-Step Registration Process

```bash
// Access EvoNext web application
https://evonext.app

// Or download the desktop application
https://github.com/sansbankdao/evonext-desktop
```

Once you're in the application:

1. Select "Create Identity" from the main menu
2. Enter your seed phrase when prompted
3. Wait for identity confirmation on the blockchain (typically 1-2 minutes)
4. Navigate to the "Username Registration" section
5. Enter your desired username and confirm availability
6. Approve the transaction to register your username

:::note Registration Fees

Identity creation and username registration both require network fees. These are paid to the network for processing your transactions and are not collected by EvoNext.

:::

### Registration Process

```bash
// Access the username registration in EvoNext
https://evonext.app/claim

// Or use the command-line interface
evonext-cli claim --username "your-chosen-username"
```

:::tip Username Selection Tips

- Usernames are case-insensitive but will display exactly as you type them
- Only alphanumeric characters are supported
- Usernames between 3-20 characters are currently supported
- Consider your long-term digital presence when choosing

:::
