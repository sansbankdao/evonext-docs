## Getting Started

### Accessing Profile Settings

1. Log in to your EvoNext account
2. Click on your profile icon in the top right corner
3. Select "Profile Settings" from the dropdown menu

### Setting Your Profile Picture

<div class="icon-grid grid-cols-2">

      <li>Verify file size (under 5MB recommended)</li>
      <li>Try refreshing the page after upload</li>
      <li>Clear browser cache and try again</li>
    </ul>
  </div>
  <div class="icon-card">
